# Fonts

Google Fonts package is used, so no manual font files needed.
The fonts will be downloaded automatically at runtime.

If you want to bundle fonts offline, download Poppins from:
https://fonts.google.com/specimen/Poppins

Required variants:
- Poppins-Regular.ttf
- Poppins-Medium.ttf
- Poppins-SemiBold.ttf
- Poppins-Bold.ttf
